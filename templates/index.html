มาตรฐาน 6 ข้อที่ระบบจัดการให้

  1. ฟอนต์ TH Sarabun New ครบ 4 slot — ascii / hAnsi / cs / eastAsia
  2. Language tag ครบ 3 ค่า — val=th-TH / eastAsia=th-TH / bidi=th-TH
  3. Complex Script flag <w:cs/> — ใส่ในทุก run ที่มีตัวอักษรไทย
  4. แยก run ตามภาษา — run ที่ผสมไทย+อังกฤษถูกแยกออกตาม script
  5. Alignment thaiDistribute — แปลง Justify เป็น thaiDistribute สำหรับย่อหน้าภาษาไทย
  6. noProof (เลือกใช้) — ปิดเส้นใต้สีแดง spell-check
